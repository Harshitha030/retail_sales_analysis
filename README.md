Retail Sales Analysis System

📌 Project Overview

The Retail Sales Analysis System is a data analytics project designed to analyze large-scale retail transaction data and generate meaningful business insights. The project uses **PySpark** for data processing and **Power BI** for interactive visualization.

---------------------------------------------

🚀 Objectives

* Analyze retail sales data efficiently
* Identify product performance and trends
* Understand customer purchasing patterns
* Generate insights for business decision-making

---------------------------------------------

🧰 Technologies Used

* Python
* PySpark
* Jupyter Notebook
* Power BI

---------------------------------------------

📂 Dataset Features

The dataset contains:

* Transaction_ID
* Order_Date
* Product_Name
* Category
* Unit_Price
* Quantity
* Total_Sales
* City
* Payment_Method

-----------------------------------------------

⚙️ Project Workflow

1. **Data Collection**

   * Retail sales dataset in CSV format

2. **Data Processing (PySpark)**

   * Data cleaning and transformation
   * Date formatting and feature engineering
   * Calculation of Total Sales

3. **Data Analysis**

   * Product-wise sales
   * City-wise performance
   * Category analysis
   * Payment method insights
   * Monthly trends

4. **Data Visualization (Power BI)**

   * Multi-page interactive dashboard
   * KPI cards and advanced visuals
   * Filters (slicers) for interactivity

-----------------------------------------------------

📊 Dashboard Pages

1. Sales Overview
2. Product Analysis
3. Category Analysis
4. City Analysis
5. Payment Analysis
6. Trends & Insights

----------------------------------------------------

📈 Key Features

* Interactive dashboard
* Multi-page analysis
* Advanced visualizations (Treemap, Ribbon, Area, Map)
* Real-time filtering using slicers

▶️ How to Run

1. Install required libraries:

```
pip install pyspark notebook pandas
```

2. Run Jupyter Notebook:

```
jupyter notebook
```

3. Execute the notebook file:

```
Retail_Sales_Analysis.ipynb
```

4. Load the processed CSV file into Power BI:

```
cleaned_sales_spark.csv
```
